Encryption

Encryption Key Terms

Cipher - The algorithm which is used to encrypt or decrypt

Plaintext - The unencrypted text

Ciphertext - The encrypted text


Symmetric Encryption

The same key is used for encryption and decryption.


Caesar Cipher

The caesar cipher is what is known as a substitution cipher. It replaces a plaintext letter with another letter which has been shifted by a set number. The number is the key.

Frequency analysis can be used to crack caesar cipher, as well as this the word structure is not changed this means that it’s easy to guess words like the.


Vernam Cipher / One Time Pad

Uses a randomly generated binary key, this key is the same lengths as the message. The message is them XOR’ed with the plaintext to produce the ciphertext.

Input:   001100101

Key:     011001010

Output:  010101111

It has many advantages over caesar, one of which is that it is perfectly mathematically secure. There is no physical way to crack the code. You will be able to get many different plaintext answers however there is no way to tell which message is which. There is no way to attack it with frequency as each bit is randomly encoded.

The one main issue is how to send the key. Since these are symmetrical the receiver will also need the key in order to decode it.
