Databases

SQL - Structured Query Language

Is a very useful thing to know! Comes up everywhere, even if your never going to sue it. It has been around for so long that the chances of it ever disappearing are tiny.


What is SQL?

SQL is a declarative programming language.

This means that we state WHAT we want rather than HOW we want to obtain the data. In SQL very little goes a long way.

SQL is a ‘guidebook’ of sorts and there are many different implementations of SQL. The one we are using is MySQL.


DDL - Data Definition Language

Used to create and maintain structures to hold data.

DML - Data Manipulation Language

Used to edit the data itself, for example editing removing adding entries to a database.


SQL Statements

USE


    USE hogwarts;

CREATE


    CREATE DATABASE hogwarts;

    CREATE TABLE students (
        `StudentNumber` INT(8) PRIMARY KEY,
        `StudentName` VARCHAR(32),
        `DateOfBirth` Date,
    );

INSERT INTO


    INSERT INTO students VALUES (00000001, 'Hermione', '1984-10-22');

    INSERT INTO students VALUES 
        (0001, 'Hermione', '1984-10-22'),
        (0002, 'Harry', '1984-09-22'),
        (0003, 'Ron', '1984-11-02');

UPDATE


    UPDATE students SET DateOfBirth = '1984-09-22' WHERE StudentName = 'Hermione';

    -- Change all entries in table students to have DOB of '1984-09-22'
    UPDATE students SET DateOfBirth = '1984-09-22';

DELETE


    DELETE FROM students WHERE StudentName = 'Hermione';

    -- Again can be made to affect all entries
    DELETE FROM students;

SELECT


    SELECT * FROM students;

    SELECT * FROM students WHERE StudentName = 'Harry';
    
    SELECT StudentNumber FROM students WHERE StudentName = 'Harry';
    
    -- Can have conditions for example
    SELECT * FROM students WHERE StudentNumber > 2;
    
    -- You can sort the result of the query ASC / DESC
    SELECT * FROM students ORDER BY StudentNumber DESC;

Will return a record set, this is a list of records with fields. Or not if you’ve done it wrong.



SQL Conditions

AND


    SELECT * FROM orders WHERE ItemCode = 2 AND OrderCode = 3;

OR


    SELECT * FROM orders WHERE ItemCode = 1 OR ItemCode = 8;

NOT


    SELECT * FROM orders WHERE NOT ItemCode = 1;

IS NULL


    SELECT * FROM orders WHERE signature IS NULL

BETWEEN... AND...


    DELETE * FROM orders WHERE OrderDate BETWEEN '2010/01/01' AND '2020/01/01';
