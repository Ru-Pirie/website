Access Modifiers

These allow is to change what parts of the code can access other parts.


Definition

Access modifiers are used to restrict code elements from use by other parts of the code.

They can be applied to a class, variable and subroutine / function definitions.


Public

This means that this can be accessed from anywhere within the code.

Private

This means that the method / variable / class can only be accessed from within the class itself. This makes it not possible to be accessed outside of the parent class.


Why go private?

It hides the information that doesn't need to be known (Abstraction!)

It also allows the programmer to control how objects properties are accessed and used. For example if you only want a variable to increased by +1 at at time not randomly set to a value.

                    
    public class TheClass {
        private int variable;
        public int getVariable() {
            return variable;
        }
    
        public void incrementVariable() {
            variable++;
        }
    }
